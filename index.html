Organizing Tests

  • tests for functions belong to "ext/standard/tests"
  • extensions tests are in "ext/standard/<module>/tests

Naming conventions

  • <functionname>_basic.phpt
  • <functionname>_variations.phpt
  • <functionname>_error.phpt
  • bug<bugid>.phpt
  • <extname><no>.phpt

Number suffix for duplications

Structure of a test


--TEST--
strtr() function - basic test for strtr()
--FILE--
<?php
/* Do not change this test it is a README.TESTING example. */
$trans = ["world" => "PHPUGMS"];
var_dump(strtr("# Hello, world! #", $trans));
?>
--EXPECT--
string(19) "# Hello, PHPUGMS! #"

--EXPECTF-- Example


--TEST--
Testing str_shuffle.
--FILE--
<?php
$s = '123';
var_dump(str_shuffle($s));
var_dump($s);
?>
--EXPECTF--
string(3) "%s"
string(3) "123"
